Shiwu的金枪鱼

金枪鱼是一款基于Revit开发的插件，它的存在是为了帮助我们更快的开发Revit样板

目前支持Revit版本 2018.2-2023




功能

过滤器模块

当前视图过滤器

  • 添加过滤器：可以选择项目中已有的过滤器应用到当前视图中
  • 移除过滤器：把选择的过滤器从当前视图移出

    • Tip : 此功能支持多选进行过滤器可见性的修改；如果当前视图已经应用了一个视图样板，那对改功能的所有操作都将会修改视图样板过滤器

复制其他视图过滤器

  • 复制过滤器：根据你所选择的其他视图的过滤器，复制到当前视图，同时会复制过滤器的可见性设置

资源模块

材质管理

  • 创建材质：在当前项目中创建一个材质，与原生功能不同的是，使用本功能创建材质的时候，会一同创建一个材质资源，与材质同名
  • 编辑材质：修改当前所选择的材质属性
  • 复制材质：复制当前所选择材质
  • 删除材质：批量删除所选择的材质
  • 导入材质：开发中...
  • 导出材质：开发中...

复制视图

  • 复制视图：在Revit中选择视图节点后，点击该功能，并修改所选择的视图的属性。

    • Tip : 所选择的视图应是同层级下的视图

风管系统

  • 创建系统：根据输入的参数创建系统
  • 编辑系统：修改当前所选择的系统属性
  • 删除系统：删除当前所选择的系统
  • 复制系统：复制当前所选择的系统
  • 创建过滤器：根据你所选择的系统，在项目中创建视图过滤器，过滤器名称与视图名称相同

管道系统

    同风管系统

桥架类型

  • 复制桥架：复制你所选择的桥架以及配件类型

    • Tip ：桥架配件将与桥架类型名称相同

  • 删除桥架：删除你所选择的桥架以及配件类型
  • 创建过滤器：根据你所选择的桥架类型，在项目中创建视图过滤器

垃圾清理

过滤器

  • 删除：批量删除不需要的过滤器

线型图案

  • 删除：批量删除不需要的线型图案
